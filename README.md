#  Quiz & Performance Tracker

A console-based quiz application developed using Python that allows users to attempt subject-wise quizzes, track scores, and analyze performance.

##  Features

- Subject-wise quizzes
- Score calculation
- Percentage analysis
- Weak-area identification
- Result storage using file handling
- View previous quiz attempts and performance history

## 🛠 Tech Stack

- Python

## 📌 Project Overview

This application helps users test their knowledge through quizzes and track their performance over time.

The system calculates scores, analyzes results, identifies weak areas, and stores previous attempts for future reference.

- Python Programming

- Conditional Statements
- Loops
- File Handling
- Data Processing
